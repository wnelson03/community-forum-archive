Questions/Help with Examples C++

I have seen a lot of threads talking about Library errors, eg:

Library_x64 C1047 C++
I would like to know, which library could I replace the library_x64, I’m using visua lstudio 2022
Library_x64 error C++
i get error when i add key auth : C1047: library_x64.lib create by different version of compiler

This is a well known “issue” (?), the Library KeyAuth uses is PUBLIC and can be found here: GitHub - KeyAuth/keyauth-cpp-library: KeyAuth C++ Library

To fix the VS error C1047 all you got to do is recompile the Library yourself with the same settings as the project you are using it for.

Now if you want to use the Library for x86 projects theres a tutorial here:

GitHub

GitHub - KeyAuth/keyauth-cpp-library: KeyAuth C++ Library

KeyAuth C++ Library. Contribute to KeyAuth/keyauth-cpp-library development by creating an account on GitHub.

As theres some people not knowing that and asking about it here I had to make a thread real quick, as always have a nice day.
