After the user logs in, you can take the string that contains the key and write it to a file using the File class of System.IO, File.WriteAllText(keystring, "key.txt");, and at the app startup you can do string savedKey = File.ReadAllText("key.txt"); to retrieve the file content and login with it. For example KeyAuthApp.license(savedkey);

Complete Code:

//After Login Code
if(!File.Exists("key.txt"))
{
   File.WriteAllText(keyString, "key.txt");
}
// App startup code
if(File.Exists("key.txt") && !String.IsNullOrEmpty(File.ReadAllText("key.txt")))
{
   string key = File.ReadAllText("key.txt");
   KeyAuthApp.license(key);
}
