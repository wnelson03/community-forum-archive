mak September 11, 2022, 3:57pm #2

Variables don’t auto-delete. Perhaps someone was able to get your seller key.

I’m seeing a lot of IP addresses using your seller key to get variable.

I’m presuming this means you put your seller API link in your program. This is very bad and will allow a user to delete users, variables, etc.

You can get variable with the KeyAuth function KeyAuthApp.var("varNameHere");

Or if you want to make request to KeyAuth Seller API to use a functionality that doesn’t exist in the KeyAuth code class, you should use the KeyAuth webhook function to hide the seller key from your users.
